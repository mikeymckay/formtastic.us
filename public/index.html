formtastic.us

Formtastic.us is a form designer for capturing data using html and javascript. Considering it an open source version of Infopath or an xforms editor/renderer will probably leave you disappointed, but it does give an indication of the formtastic.us problem space (and future direction). Formtastic.us creates html forms with validation and field level form logic and can even render forms with a touchscreen interface.

Try it

The source is available from our git repository:
git clone git://baobabhealth.org/formtastic.git
Formtastic.us uses ruby on rails, prototype, scriptaculous, rspec, and BDD for javascript, the OpenMRS data model and the touchscreen toolkit.
