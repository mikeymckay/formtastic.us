formtastic.us

Formtastic.us is a form designer for capturing data using just html and javascript. Considering it an open source version of Infopath or an xforms editor/renderer will probably leave you disappointed, but it does give an indication of formtastic.us's problem space. If you need a tool to make html forms with validation and form logic and even touchscreen interfaces then formtastic.us might be just what you are looking for. Formtastic uses the OpenMRS data model for capturing the form details.

Try it

Formtastic.us uses ruby on rails, prototype, scriptaculous, rspec, and BDD for javascript.
